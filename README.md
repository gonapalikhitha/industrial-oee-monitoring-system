# Industrial OEE Monitoring System

## Overview
This project is an automated industrial OEE monitoring and reporting system developed using Python.

The system reads production data from CSV files, calculates OEE metrics, performs downtime analysis, stores reports into SQLite database, generates professional HTML dashboards, and sends automated email reports.

---

## Features

- Automated OEE Calculation
- Availability, Performance, Quality Analysis
- Downtime Analysis
- HTML Dashboard Generation
- SQLite Database Integration
- Automated Email Reporting
- Task Scheduler Automation

---

## Technologies Used

- Python
- Pandas
- SQLite
- HTML/CSS
- Chart.js
